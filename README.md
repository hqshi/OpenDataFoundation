# OpenDataFoundation

## 便利 公正的GTA5游戏数据查询服务
**极速查询数据，作弊无处遁形**
> 便利: 免驱繁琐登录，快速查看游戏数据和账号记录。
> 
> 公正: 对任何玩家均一视同仁，不做庇佑。任何人都可以使用，不受任何限制。





支持平台

* 微信
* 1. 微信公众号 GTA5 游戏助手
* 2. 微信小程序 洛圣都 Express
* QQ群机器人 因腾讯限制，暂时不提供支持
* Api 暂未开放
