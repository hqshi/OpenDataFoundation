# OpenDataFoundation

## 便捷、客观的GTA5游戏数据查询平台
支持平台

* 微信
* 1. 微信公众号 GTA5 游戏助手
* 2. 微信小程序 洛圣都 Express
* Api 暂未开放
* QQ群机器人 因腾讯限制，暂时不提供支持
